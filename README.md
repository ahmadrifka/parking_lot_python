# Parking Lot App


Parking lot app is a command-line parking game build with python

## Features

- You can create your own parking lot with total slots
- Park your car with your license number 
- Remove your car from parking 
- And see where's parking slot empty

## Installation

Parking Lot App requires [Python](https://www.python.org/download/releases/3.0/) to run.

Clone this repository and go to parking_lot directory.  Then : 

```sh
cd parking_lot
python main.py
```
